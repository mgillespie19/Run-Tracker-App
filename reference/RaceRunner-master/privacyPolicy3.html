Privacy Policy

RaceRunner uses the Amazon Web Services™ analytics product, Pinpoint™, to track certain information. This information consists of:

  • Each screen visited
  • The act of starting a run
  • The act of pausing a run
  • The act of resuming a run
  • The act of finishing a run
  • App version
  • Unique installs
  • iPhone model
  • Country of user

RaceRunner does not track any personally identifiable user information.

Please direct any questions about or feedback on this privacy policy to RaceRunner's developer.
